Minimum working example for Manuscript: "No Model Fits All: Dynamic Ensemble Species Distribution Model Reveals Seasonal Patterns of Essential Habitat Use of Ocean Giants in the Southwest Pacific"

It contains the core R code used to build the species distribution models, including a set of helper functions that streamline the dynamic modelling workflow.


Miller, I.B., Niella, Y., Udyawer, V., Erdmann, M.V., Abrantes, K., Pierce, S., Fitzpatrick, R., Hoopes, L., Dove, A.D.M., & Barnett, A. (in press). No Model Fits All: Dynamic Ensemble Species Distribution Model Reveals Seasonal Patterns of Essential Habitat Use of Ocean Giants in the Southwest Pacific. Diversity and Distributions. https://doi.org/https://doi.org/10.1111/ddi.70186
